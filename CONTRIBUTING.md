## Pesan Commit

Diharapkan untuk menulis pesan commit dengan format berikut.

| Format    | Keterangan                     |
| --------- | ------------------------------ |
| `add:`    | jika ingin menambah sesuatu    |
| `change:` | jika ingin merubah sesuatu     |
| `fix:`    | jika ingin memperbaiki sesuatu |
| `rm:`     | jika ingin menghapus sesuatu   |
